# KirillYemetsNotes

Pet project for making notes created with **Jetpack Compose** and **Firebase**.

### Existing features: 
- [x] Add notes
- [x] Save notes
- [x] Delete notes with swipe
- [x] Synchronize with cloud (using firebase)
- [x] Mark favorite notes
- [x] Search notes by text
